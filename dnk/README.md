## Description
### Игры с ДНК
Биолог изучает сегмент ДНК, связанный с генетическими заболовенаиями.  Тот состоит из 23 пар нуклеотидов, где 9 должны быть А-Т, а 14 - G-C
Ученый хочет выполнить моделирование на всех возможных сегментах ДНК, где есть такое количество пар нуклеотидов. Сколько задач ему предстоит выполнить?
