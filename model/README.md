## Description
### Совершенная мелодия
Девушка разучивает гаммму из 13 нот. Она хочет, чтобы вы  показали все возможные мелодии, в которых используется 6 нот. Каждая нота должна встречаться один раз за мелодию, а каждая такая мелодия должна звучать в течении одной секунды. О какой продолжительности звучания идет речь?
